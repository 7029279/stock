# stock
株ロボのデイトレシミュレーター作ってみました。

分足時足データが手に入らないので日足データを使用します。基本始値で買って終値で売ると仮定します。

銘柄は前日１５時以降か当日に適時公開の業績予想修正が出ているものの中で業績が好調なもの（１株ごとの純利益が高い）を選びました。

５０００円から初めて３ヶ月弱で７２４０円 (一日平均0.0043125＋)

結論：そんなにもうからない。分足データがあればロスカットを細かく組めるのでもっと儲かるかもしれません。あと臨時の業績修正だけでなく四半期決算に書かれている情報も判断に加えれば利益率が向上するはずです。時間あったらやろうと思います。
